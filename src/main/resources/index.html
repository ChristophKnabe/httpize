httpize(1): HTTP Request & Response Service

ENDPOINTS

  • / This page.
  • /ip Returns Origin IP.
  • /user-agent Returns user-agent.
  • /headers Returns header dict.
  • /gzip Returns gzip-encoded data.
  • /cookies Returns cookie data.
  • /cookies/set?name=value Sets one or more simple cookies.
  • /cookies/delete?name Deletes one or more simple cookies.
  • /html Renders an HTML Page.
  • /robots.txt Returns some robots.txt rules.

DESCRIPTION

Testing an HTTP Library can become difficult sometimes. Postbin is fantastic for testing POST requests, but not much else. This exists to cover all kinds of HTTP scenarios. Additional endpoints are being considered.

All endpoint responses are JSON-encoded.

NOTE: Inspired from httpbin.org

EXAMPLES

$ curl http://httpize.herokuapp.com/ip

    "remote": "10.34.186.222",
    "x-forwarded-for": "Some(X-Forwarded-For: 37.81.65.193, 37.81.65.193)"
}
